Detailed Description

WriteSubWindowDockPlugin.

This is an example of dock for the WriteSubWindow, in the "Write" panel. This is a minimal dock with two parts : Core and Gui . The Gui part calls a writetab_dock.ui form. The compiled form writetab_dock_ui.py is build automaticaly if the environement variable PLUME_DEVELOP_FROM points to a parent directory of this plugin.

The goal of this dock is to display and modify the title of the current sheet. Nothing really useful...
